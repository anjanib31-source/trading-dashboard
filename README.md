# ALPHA Trading Bot

> Automated trading system for Indian equity markets using Angel One API

## 🚀 Features

- **Multi-Strategy Trading**: ORB, VWAP, Trend Following
- **Risk Management**: Dynamic SL, ATR sizing, Sector diversification
- **Real-time Data**: WebSocket with heartbeat, Yahoo Finance fallback
- **Monitoring**: Health checks, metrics, Telegram alerts
- **Dashboard**: Web UI with charts, analytics, and health status
